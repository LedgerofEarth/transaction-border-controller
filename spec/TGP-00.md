# TGP-00: Transaction Gateway Protocol

**Status:** Draft  
**Version:** 0.1-draft  
**Organization:** Ledger of Earth  

Defines the Layer-8 control-plane protocol for routing, negotiating, and
settling cross-ledger value flows across trust boundaries.

Integrates with:
- TxIP-00 (transaction signaling)
- X402-EXT (payment session layer)
